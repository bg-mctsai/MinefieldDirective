# 🗺️ 100 關全戰役規劃書 (100-Level Strategic Campaign)

## 0. 核心邏輯調整
* **數字難度係數**：
    * **簡單 (Level C)**：`3`, `4`, `5`, `6` (周圍空間大，容錯高)
    * **中等 (Level B)**：`2`, `7` (邏輯點變窄)
    * **困難 (Level A)**：`1`, `8` (極度精確，1 代表周圍只能有一顆雷，8 代表必須全滿)
* **過關條件**：
    * **覆蓋率 (Coverage)**：必須正確填滿地圖指定比例的格子。
    * **邏輯一致性**：所有已填數字必須完全符合邏輯，不可爆炸。

## 0.1 關卡挑戰與解鎖規則
* **關卡總數固定**：`1..100`（共 100 關）。
* **初始解鎖**：新存檔只解鎖 `Level 1`；`Level 2..100` 預設鎖定。
* **線性解鎖**：通關 `Level N` 後更新進度 `highestClearedLevel = max(highestClearedLevel, N)`，並解鎖 `Level (highestClearedLevel + 1)`（若 `<= 100`）。
* **不可跳關**：UI 與進入戰鬥流程都必須檢查解鎖條件；未解鎖關卡不可進入戰鬥。
* **存檔防呆**：讀取進度若異常，需 `clamp` 到 `0..100`。

## 1. 章節演進詳細表

### 第一章：新兵訓練營 (Lv.1 - 10)
* **地圖**：方形 / 長方形 (5x5, 6x6, 8x8)。
* **格數**：25 ~ 64 格。
* **指令範圍**：僅出現 `3`, `4`, `5`, `6`。
* **過關門檻**：覆蓋率 70%。
* **目標**：熟悉基本的「逆向佈雷」手感，無時間壓力。

### 第二章：城市巷戰 (Lv.11 - 20)
* **地圖**：多邊形、菱形、十字形。
* **格數**：64 ~ 100 格。
* **指令範圍**：加入 `2`, `7`。
* **過關門檻**：覆蓋率 70%。
* **目標**：應對非對稱的地圖邊緣，邊緣格子的邏輯判斷會變複雜。

### 第三章：沙漠閃擊 (Lv.21 - 35)
* **地圖**：方形 + 障礙物（不可佈雷區）。
* **指令範圍**：解鎖 `1`, `8` (極難指令)。
* **新機制**：**解鎖新角色（工程師）**、**開始計時 (60s~120s)**。
* **過關門檻**：覆蓋率 75%。
* **目標**：在時間壓力下處理極端數字。

### 第四章：三角形陣地 (Lv.36 - 50)
* **地圖**：**三角形網格結構** (每個格子周圍僅有 3 或 12 個鄰居，視鑲嵌方式而定)。
* **指令範圍**：全數字隨機。
* **新機制**：**解鎖新角色（老兵）**。
* **過關門檻**：覆蓋率 75%。
* **目標**：改變九宮格邏輯，適應三角形的鄰近關係。

### 第五章：蜂巢幽閉 (Lv.51 - 70)
* **地圖**：**六角蜂狀結構 (Hexagon)**。
* **指令範圍**：出現 0 ~ 6 (六角形鄰居上限為 6)。
* **新機制**：**意外事件：通訊干擾** (手牌數字隱藏)。
* **過關門檻**：覆蓋率 80%。
* **目標**：蜂巢結構無死角，邏輯環環相扣，錯一格全盤炸裂。

### 第六章：異次元裂縫 (Lv.71 - 90)
* **地圖**：混合地形（方形、三角形、六角形在同一關出現）。
* **指令範圍**：全數字隨機，出現「偽裝指令」。
* **新機制**：**解鎖最終角色（情報官）**。
* **過關門檻**：覆蓋率 85%。
* **目標**：極限邏輯切換，考驗對不同幾何結構的適應力。

### 第七章：終焉防線 (Lv.91 - 100)
* **地圖**：超大型複雜形狀（如骷髏、閃電、國境線形狀）。
* **指令範圍**：大量 `1` 與 `8`。
* **新機制**：**雙重意外事件**（計時 + 迷霧）。
* **過關門檻**：覆蓋率 90%。
* **第 100 關**：名為「神之眼」，全地圖 100% 覆蓋要求，無容錯。

## 2. 數據結構參考 (JSON 結構)
為了讓 Cursor 幫你生成關卡，建議定義如下格式：

```json
{
  "level": 25,
  "chapter": 3,
  "mapType": "SQUARE_WITH_OBSTACLES",
  "gridType": "RECTANGLE", 
  "config": {
    "size": { "x": 10, "y": 10 },
    "targetCoverage": 0.75,
    "timeLimit": 90,
    "allowedInstructions": [1, 2, 3, 4, 5, 6, 7, 8],
    "specialEvent": "SANDSTORM"
  }
}
