# 遊戲核心邏輯 (Core Logic)

## 1. 基礎規則 (The Core Loop)
- **佈雷機制**：玩家選擇手牌數字 -> 點擊地圖 -> 該格放置地雷。
- **邏輯判定**：每個有地雷的格子會檢查周圍 8 格的地雷總數是否等於該格「數字」。
- **失敗條件**：一旦玩家的操作導致任何一格出現「邏輯矛盾」（例如：數字為 3，周遭卻有 4 顆雷），觸發「連鎖爆炸」。

## 2. 遊戲分頁結構
- **Home (首頁)**：入口動畫、戰略指南。
- **Hero Select (英雄選擇)**：解鎖與選擇不同技能的角色。
- **Mission Map (關卡地圖)**：固定 `1..100` 關卡的線性闖關進度（只能從第 1 關開始逐關挑戰）。
- **Battlefield (戰鬥畫面)**：核心遊戲操作區。

## 3. 指令系統 (Commander's Orders)
- 玩家每回合擁有的手牌數字是由系統隨機生成的「合規數字池」。
