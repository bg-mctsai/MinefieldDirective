# 📄 遊戲首頁企劃書 (Home Page Specification)

## 1. 視覺風格與氛圍 (Visual & Atmosphere)
- **核心主題**：軍事戰術終端機 / 高科技暗黑風格。
- **配色方案**：
  - 背景：`#0B0E14` (深煤黑)
  - 主色（動作）：`#F59E0B` (螢光橘 / 琥珀色)
  - 副色（安全）：`#10B981` (電子綠)
  - 警告（衝突）：`#EF4444` (警示紅)
- **特效需求**：
  - 全屏背景帶有淡淡的「掃描線」 (Scanlines) 濾鏡。
  - 背景隱約可見雷達掃描動畫 (Radar Sweep)。
  - UI 組件進入時應有「系統加載」的切換感。

---

## 2. 頁面組件架構 (Component Tree)

### A. 頂部標題區 (Header)
- **遊戲標題**：`雷區指令：戰略部署` (採用特粗無襯線字體，帶有打字機緩緩出現的動畫)。
- **副標題**：`[系統已就緒] 指揮官，請下達埋雷指令。`
- **狀態列**：顯示當前版本號 `v1.0.4` 與系統連線狀態 `CONNECTED`。

### B. 中央主選單 (Navigation Menu)
每個按鈕在 Hover 時應有「發光」效果與輕微的音效。
1. **[ 開始行動 ] (Mission Start)**
   - 功能：進入關卡選擇地圖 (`MissionMap`)。
   - 樣式：大尺寸橘色按鈕，呼吸燈特效。
2. **[ 角色整備 ] (Hero Gallery)**
   - 功能：進入角色技能查看與選擇頁面 (`HeroSelect`)。
3. **[ 戰略指南 ] (Strategic Manual)**
   - 功能：彈出 Modal 顯示規則說明。
4. **[ 系統設定 ] (Settings)**
   - 功能：調整音量、清除存檔、語言設定。

## 3. 戰略指南詳細內容 (Strategy Manual Content)
當點擊 [戰略指南] 時，以分頁標籤形式顯示：

### 🔖 指南一：核心邏輯
- **身份**：你不是在排雷，你是在「佈雷」。
- **邏輯**：地圖上的數字 $N$ 代表該格周圍 8 格（九宮格範圍）必須且只能有 $N$ 顆地雷。
- **連鎖反應**：一旦佈署導致邏輯矛盾，全線地雷將立即產生連鎖爆炸，任務失敗。

### 🔖 指南二：操作流程
- **選取指令**：從右手邊的「長官指令」中挑選一個數字。
- **派遣小兵**：點擊地圖方格，小兵會移動到該處埋雷。
- **達成目標**：滿足地圖上所有數字要求，並達到要求的「覆蓋率」即可過關。

## 4. 角色快速展示區 (Hero Spotlight)
- 在選單旁邊顯示當前選中的角色頭像（或剪影）。
- **動態效果**：角色周圍有數據流（Data Stream）上下穿梭。
- **對話框**：角色會隨機說出一句台詞（例如：「這場戰爭的勝負，在於你的計算。」）。

---

## 5. 開發實作重點 (Implementation Notes)
- **動畫庫**：建議使用 `framer-motion` 處理 Page Transition。
- **字體**：選用等寬字體 (Monospace) 增加科技感。
- **Responsive**：
  - Desktop：選單位於左側，角色展示位於右側。
  - Mobile：垂直堆疊，角色展示移至背景或縮小。
