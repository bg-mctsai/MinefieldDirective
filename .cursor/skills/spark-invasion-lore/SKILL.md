# 星火入侵世界觀（Spark Invasion Lore）

**劇情主軸**亦稱《**雷區指令：星火絕境**》。目標：讓戰役／簡報／系統敘事共用**同一套**異界入侵設定與專有名詞，避免多頭馬車。

## 敘事核（V2 對齊）

- **環境**：極壓星際開採與裂隙周邊防線。
- **敵方**：噬星者——**動態視力與防禦極強**、**靜態盲區**；非常規對射取勝。
- **我方戰術**：以玩家熟悉的**地雷／佈雷／雷區**為主詞；**停置休息期**內**緩慢、靜默**接近並完成靜態佈署。
- **物理解釋**：需要時可帶星火裝藥、觸發條件、頻率或節點等**技術語氣**，**不**強迫用新名詞取代「地雷」。

## 章節敘事弧（戰役規劃）

長稿與逐章細節：**[`ref/chapter-narrative-arc.md`](ref/chapter-narrative-arc.md)**（核心事件、世界觀錨點、玩家課題、情感基調、終章三向規劃、快速參考表）。

**階段總覽**（與 `emotion-route-balance` 並用時，先對齊本章「嚴厲／關懷／混合」意圖）：

| 章節   | 階段        | 敘事重心（摘要） |
|--------|-------------|------------------|
| 第 1–2 章 | 嚴厲     | 靜默戰法學習與前線初次實踐 |
| 第 3–5 章 | 混合     | 據點防守、休息期認知、幾何／蜂巢與系統性取捨 |
| 第 6–7 章 | 嚴厲     | 電報／洩漏／干擾、信任崩塌 |
| 第 8–9 章 | 關懷     | 體制外同盟、倫理與共振技術線、希望回溫 |
| 第 10 章 | 嚴厲＋關懷 | 終局三向度、開放式結論 |

## 進階敘事與 glossary 對齊（長弧專用）

`chapter-narrative-arc` 允許**人類派系、洩漏、談判拖延、利益交換**等劇。撰寫時仍須遵守：

- **與誰談判**：終局「關懷路線」等**談判、交易、拖延**的對象須為**人類／組織**，**不是**與噬星者議和或締約。
- **噬星者的「智」**：可寫**蜂群式湧現、路徑記憶、壓迫升級**，不可升格為**有使節、有外交、可溝通的文明主體**（見 `glossary`）。
- **被操控**：若寫利用／誘導噬星者，應落點在**人類如何操縱能量與礦區分布**，避免暗示「外星帝國幕後操線」與掠食者設定打架。
- **天災感保留**：權鬥再黑，噬星者仍是**生態級掠食災害**；避免寫成單一陰謀一招就徹底終結威脅，除非明講這是某一結局分支的敘事取捨。

## 與本專案其他內容的關係

- **玩家角色**：敘事上固定為**工匠（Artisan）**；實作上是否具名、介面用語可與「前沿作戰幹員」等 UI 文字並存，但世界觀層級以工匠為準。
- **玩法詞**（長官電報、佈雷、雷區、鄰焰共振等）屬**操作／機制層**；本 skill 的**世界觀層**以 **地雷＝靜態佈署、噬星者＝動態強／靜態弱** 為預設解釋，與 UI 用語一致。
- 撰寫須**可貼入 JSON 的實作文案**（進場簡報、固定訊息等）時，口氣與規則可讀性另遵守：`.cursor/skills/emotion-route-balance/SKILL.md`；**低調、靜默、潛行佈署**類台詞規範見 [05_dialogue_voice_guide_2.md](../../../steam-win/docs/planning/05_dialogue_voice_guide_2.md)。

## 目前範圍外（刻意不寫死）

- **可變角色的動態回饋台詞**（如各幹員 bark、戰況情緒句）：人選與台詞可能重調，不收入本 skill。
- **關卡地圖、據點形狀、章節戰役路線**：見 `.cursor/skills/campaign-map-themes/SKILL.md`。
- **關卡時間、難度、單關流程推演**：見 `.cursor/skills/level-time-adjustment/SKILL.md` 或專案文件。章節「應有」的戲劇張力見 `ref/chapter-narrative-arc.md`，但**與關卡 JSON 一一對應**須另維護。

## Gotchas

- 把「星火」寫成無限能源神棍設定，會與**開採風險、補給與邊界收束**的壓迫感衝突。
- 把噬星者寫成有文明、有談判餘地，與**純粹掠食**的設定衝突。
- 把**主力戰法**寫成衝鋒槍砲取勝，與**動態火力低效、靜態地雷為反擊主軸**衝突。
- 在簡報裡硬塞教學規則卻漏掉情緒——應交給 `emotion-route-balance` 的「規則與情緒並存」原則，本 skill 不取代玩法說明。
- 長弧若寫「終極電報馴服／休眠噬星者」：宜保持**頻率—生理／生態反應**的技術語氣，避免寫成**與異界智慧文明溝通成功**。
