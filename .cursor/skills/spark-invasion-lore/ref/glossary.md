# 名詞定義（Glossary）

下列為《**雷區指令：星火絕境**》**星火—噬星者**敘事軸的優先用語。英文括號供 UI／設定稿對照，正文以繁中為主。

| 名詞 | 定義 / 敘事用途 |
|------|-----------------|
| **星火絕境** | 作品劇情向副標，強調人類在能量被啃食、邊界收束下的戰略防線敘事；**非**額外世界線，與本表其他名詞同一宇宙。 |
| **星火（Spark）** | 人類開採、精煉之**高能礦物**；是空間傳送與工業的能源基盤，亦為噬星者**首要覓食目標**之一。**禁止**寫成無限、神蹟式能源；維持開採風險與補給壓迫感。 |
| **空間傳送技術** | 以星火為基礎開啟的**跨空間傳送**；故事上等同「撕裂現實、連通異界」的技術前提，**不是**本 skill 要細寫的科幻設定集。 |
| **異界／裂隙** | 敘事上可泛稱傳送導致之**空間不連續、通道或污染前緣**；作戰區多在裂隙影響範圍或預測路徑上。 |
| **噬星者（Star-Eaters）** | 自異界而來的**掠食性威脅**（敘事可稱高能聚合體或生物型態）；**無**單一文明、**不可**談判，以吞噬能量物質維生。對人類是**生態級災難**。 |
| **動態視力／動態防禦** | 噬星者對**高速、高動態**威脅的偵知與攔截、能量偏移能力；用於解釋「為何常規衝鋒火力不是主解法」。 |
| **靜態盲區** | 噬星者對**靜止、緩變、低動態**物體的遲鈍分類（常當成背景）；用於解釋「為何**地雷**等靜態佈署能接近或預置而不被當成首要威脅」。 |
| **停置休息期** | 噬星者大量進食星火後的**長時間靜止／低活動**階段；敘事上常為工匠**緩慢接近、靜默佈雷**的時間窗口（仍忌用動態火力驚敵）。 |
| **地雷／佈雷／雷區（介面與敘事主詞）** | 專案與玩家熟悉的用語：戰術格上**靜態佈署**與連鎖觸發的作戰手段。世界觀上可帶過星火裝藥、保險、遙控或格點觸發等**技術感**，**不要**為了「科幻感」強迫全文改叫新名詞。 |
| **共振、頻率、節點（物理解釋層，選用）** | 需要技術語氣時，可描述地雷如何在觸發時對核心造成**結構性毀傷**；與關卡 JSON 機制名並用時**勿發明另一套**取代鄰焰共振、熱力迴聲等**已鎖專名**。 |
| **長官電報（作戰層）** | 專案用語，指**上級下達的作戰數字與任務佇列**。敘事上屬**加密／壓縮的戰術指令**傳遞，與星火技術不衝突，亦可在簡報中帶**干擾、遲滯、扇區讀數**的壓迫感。 |
| **鄰焰共振、熱力迴聲（機制層）** | 專案關卡機制專用名。敘事上可聯想為能量在**已佈節點間的傳導／加成**；**不要**在 glossary 裡用無關物理名去取代，以免與關卡 JSON 不一致。 |
| **工匠（Artisan）** | **玩家戰術指揮的敘事代號**；職責是決策、佈雷與戰術執行，不預設固定姓名。 |
| **地平線防衛軍** | 所屬武裝通稱；**世界觀錨點**用。細部編制、階級、內部派系可之後單開設定，**不在本表鎖死**。 |
| **小隊／前線幹員** | 泛指工匠指揮之作戰單位。具名角色、人設與台詞**可變動**，劇情上一律可用職能稱呼（如通訊、偵查、工兵等）帶過，避免釘死。 |

## 一講就錯的別名（撰寫時注意）

- **星獸、異獸、外星人** 等籠統詞**可**口語帶到，但正式名仍以 **噬星者** 為主。
- **把噬星者寫成有帝國、有使節、有情緒的文明** — 與其「純掠食」敘事核衝突，除非專案另啟一條分支設定（目前無）。
- **把本作主戰術寫成槍戰、突擊、砲轟取勝** — 與「動態火力低效、靜態佈雷為主軸」衝突；若帶到開火，應是**負面示範或絕境下的錯誤選項**，而非官方戰法。
