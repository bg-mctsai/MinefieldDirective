# 名詞定義（Glossary）

下列為《**雷區指令：星火絕境**》**星火—噬星者**敘事軸的優先用語。英文括號供 UI／設定稿對照，正文以繁中為主。

| 名詞 | 定義 / 敘事用途 |
|------|-----------------|
| **星火絕境** | 作品劇情向副標，強調人類在能量被啃食、邊界收束下的戰略防線敘事；**非**額外世界線，與本表其他名詞同一宇宙。 |
| **星火（Spark）** | 人類開採、精煉之**高能礦物**；是空間傳送技術的能源與工業基盤，亦為噬星者**首要覓食目標**之一。 |
| **空間傳送技術** | 以星火為基礎開啟的**跨空間傳送**；故事上等同「撕裂現實、連通異界」的技術前提，**不是**本 skill 要細寫的科幻設定集。 |
| **異界／裂隙** | 敘事上可泛稱傳送導致之**空間不連續、通道或污染前緣**；作戰區多在裂隙影響範圍或預測路徑上。 |
| **噬星者（Star-Eaters）** | 自異界而來的**生物型掠食者**；多樣貌、無單一文明，以吞噬能量物質維生。對人類是**生態級災難**而非可談判的勢力。 |
| **共振裝具／共振轟毀** | 將能量化星火調至**可破壞噬星者內在核心**之共振條件後引爆；敘事上解釋「為何佈在格點上的能量作戰能毀敵」——**內部共振、擊毀核心**，而非單純從外炸碎。遊戲內可與「地雷」「佈雷」等**玩家用語**並用：前者是**物理解釋層**，後者是**介面層**。 |
| **雷區 / 佈雷（介面層）** | 專案既有 UI 與機制用語，指戰術格上**佈署與連鎖**的玩法空間。世界觀上可稱作**能量節點佈防、扇區封鎖**等，**不必**逐句改寫所有 UI 成科幻長句。 |
| **長官電報（作戰層）** | 專案用語，指**上級下達的作戰數字與任務佇列**。敘事上屬**加密／壓縮的戰術指令**傳遞，與星火技術不衝突，亦可在簡報中帶一點**干擾、遲滯、扇區讀數**的壓迫感。 |
| **鄰焰共振、熱力迴聲（機制層）** | 專案關卡機制專用名。敘事上可聯想為星火能量在**已佈節點間的傳導／加成**；**不要**在 glossary 裡用另一套完全無關的物理名去取代，以免與關卡 JSON 不一致。 |
| **工匠（Artisan）** | **玩家戰術指揮的敘事代號**；職責是決策、佈能與戰術執行，不預設固定姓名。 |
| **地平線防衛軍** | 所屬武裝通稱；**世界觀錨點**用。細篔編制、階級、內部派系可之後單開設定，**不在本表鎖死**。 |
| **小隊／前線幹員** | 泛指工匠指揮之作戰單位。具名角色、人設與台詞**可變動**，劇情上一律可用職能稱呼（如通訊、偵查、工兵等）帶過，避免釘死。 |

## 一講就錯的別名（撰寫時注意）

- **星獸、異獸、外星人** 等籠統詞**可**口語帶到，但正式名仍以 **噬星者** 為主。
- **把噬星者寫成有帝國、有使節、有情緒的文明** — 與其「純掠食」敘事核衝突，除非專案另啟一條分支設定（目前無）。
