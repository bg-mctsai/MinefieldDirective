# 世界觀背景：星火、噬星者與靜態佈雷

**作品稱呼**：《雷區指令：星火絕境》——人類在**極度高壓的星際開採**環境下，與異界捕食者「噬星者」的生存博弈；核心戰術是**利用敵方生理缺陷**，進行**靜態地雷佈署**。

## 1. 星火、傳送與入侵

未來世界，人類開採高能礦物**星火（Spark）**，並以其為基礎發展**空間傳送**技術。通道與高能作業把現實邊界撕薄，引來平行**異界**的掠食者——**噬星者（Star-Eaters）**。

噬星者**不是可談判的文明**；敘事上可視為走向極端的**高能聚合體／生物型掠食者**（體型與外觀各異），行為上只有一個目的：**吞食可取得的能量與物質**（星火及其衍生物最為敏感）。

## 2. 噬星者：動態優勢與靜態缺陷

### 動態視力與反射（牠的優勢）

噬星者具備近乎異常的**動態視覺與防禦反射**：能捕捉、攔截高速威脅（如超音速彈體或高能射束）。任何**高速、高動態**的攻擊都容易觸發其本能的**能量偏移或反擊**。

**敘事結論**：常規火力「硬衝」對其**極低效甚至招災**；戰場上**不**以槍砲對決為主軸。

### 靜態盲區（人類的破口）

其感官對**低能量、靜止或極緩慢移動**的物體極度遲鈍，常將之歸類為**環境背景**（岩塊、廢墟、未啟動的裝具）。處於**靜態**時，上述動態防禦鏈**較不易被觸發**。

**地雷**（玩家與 UI 的慣用詞）在敘事上即為此類**靜態佈署**：預置於路徑或格點，待目標**移動、踐踏或進入觸發條件**時才釋放殺傷。可依劇情需要帶過「星火裝藥／起爆條件」等技術感，**不必**用新名詞取代「地雷」二字。

### 停置休息與人類窗口

噬星者在**大量吞噬星火**後，會進入長時間的**停置休息期**以轉化能量。此時牠們看似靜止，對周遭**動態能量**仍可能保有**強烈反擊本能**——因此人類在窗口期採取**緩慢、靜默的移動**，接近預定區域並完成**地雷佈署**，而非用動態火力驚醒全場。

> 戰爭的表象是裂隙與開採線周邊的拉鋸；實際是在噬星者抵達**能源與據點**前，於其**生理盲區與時間窗口**內完成**雷區封鎖**。

## 3. 主角職責：工匠

玩家敘事身分為戰術指揮**工匠（Artisan）**，所屬**地平線防衛軍**。任務本質**不是一線衝鋒對射**，而是在壓力與時限下，於指定扇區**判讀威脅、完成靜態佈雷與戰術決斷**。具名同袍、可變台詞**不**由本文件鎖定，以免與幹員實作改動衝突。
