# 世界觀背景：異界的撕裂

**作品稱呼**：《雷區指令：星火絕境》——人類以星火與頻率作戰、在邊界被啃食殆盡前的戰略防線敘事。

## 1. 星火、傳送與噬星者

未來世界，人類發現高能礦物**星火（Spark）**，並以其為基礎發展**空間傳送**技術。技術打開的通道也引來平行**異界**的掠食者——**噬星者（Star-Eaters）**。

噬星者並非高科技文明，而是走向極端的**進化生物**：體型與外觀各異（如巨蟲、漂浮的半透明腔腸體等），沒有共通的「社會」或可交涉動機，行為上只有一個目的：**吞食一切可取得的能量性物質**（星火及其衍生物最為敏感）。

人類的常規動能、熱兵器對其**效果有限**；關鍵在於將星火**能量化**後，在特定**頻率／相結構**下引爆——**內部共振**能摧毀其生理核心。據此製成的作戰裝具在敘事上可對應遊戲內的「地雷／佈署節點」：在雷區內**按戰術節點佈能**，是對抗大規模推進的少數有效手段之一。

> 戰爭的表象是**傳送門／裂隙**周邊的拉鋸；實際是在噬星者抵達**能源與據點**前，盡力完成**毀壞其核心的頻率作戰**。

## 2. 主角職責：絕境的守望者

玩家敘事身分為戰術指揮**工匠（Artisan）**，所屬為**地平線防衛軍**。任務本質**不是一線衝鋒**，而是在敵潮壓上之前，在指定扇區內**構築最後防線**——在時間與資源 constraints 下完成佈能與戰術決斷。具體同袍名單、戰友個性與可變台詞**不**由本文件鎖定，以免與幹員實作改動衝突。
