# Lv1 情感路線可貼用稿

用途：當使用者說「先從第一關開始做嚴厲/關懷」時，直接回傳以下 JSON 片段。

## A. 章節入場（`chapterEntryBriefings.json`）

```json
{
  "byLevelId": {
    "1": {
      "strict": [
        "眼睛睜大。這裡是新兵訓練營，不是觀光區。",
        "長官會一次發多封電報過來，你給我照數字佈雷。順序錯、座標錯，就立刻重練。"
      ],
      "care": [
        "看著我，穩住。這裡是新兵訓練營，你會緊張很正常。",
        "等一下會同時收到幾封電報：先選數字，再選座標。就算失手也能重練，我要你活著學會。"
      ]
    }
  }
}
```

## B. 動態反饋（`gameFixedMessages.json`）

```json
{
  "lossCommander": {
    "digitMismatchClue": {
      "strict": "長官回電：「電報叫你埋『{v}』，盤面卻是『{w}』。別硬上，重選！」",
      "care": "長官回電：「你看到的是『{v}』，但盤面是『{w}』。先退一步，我們重選一次。」"
    }
  },
  "victoryStatus": {
    "plain": {
      "strict": "報告長官！依電報完成佈雷，任務完成。下關不准鬆懈。",
      "care": "報告長官！依電報完成佈雷，任務完成。做得好，保持這節奏。"
    }
  }
}
```

## 最小驗收

- 嚴厲版讀起來像高壓指揮；關懷版讀起來像保命導師。
- 兩版都保留同一個操作流程，不讓玩家混淆。
- 每版至少 1 條失誤回饋 + 1 條成功回饋。
