# 角色個性與說話方式（Voice Profiles）

> **用途**：撰寫長官簡報、幹員 `barrage`、作戰地圖台詞、卷宗敘事時的**人設與口氣依據**。  
> **實裝來源**：[`steam-win/renderer/src/heroes.ts`](../../../../steam-win/renderer/src/heroes.ts)（幹員）、[`chapterEntryBriefings.json`](../../../../steam-win/renderer/src/levelData/chapterEntryBriefings.json)（長官）。  
> **世界觀禁忌**：[`spark-invasion-lore/ref/glossary.md`](../../spark-invasion-lore/ref/glossary.md)。  
> **長官口氣骨架**：[`sergeant-tone-rules.md`](sergeant-tone-rules.md)。  
> **企劃摘要版**：[steam-win/docs/planning/07_character_voice_profiles.md](../../../../steam-win/docs/planning/07_character_voice_profiles.md)

## 一、全體共通原則

| 項目 | 規範 |
|------|------|
| **戰術主詞** | 佈雷、設置、覆蓋率、格點、電報、頻道、讀碼——**不宜**把官方戰法寫成衝鋒、開火、轟炸 |
| **句長** | 戰鬥中台詞偏短（8～22 字）；長官簡報可稍長但仍以短句堆疊 |
| **戰場口語** | `barrage`、對局狀態句**像嘴邊話**：可省略主語、用語助（欸、啦、嘛、喔）；專業詞當口頭習慣帶出，**不要**簡報／報告體（「結果輸出：」「邏輯上該格僅允許」） |
| **規則與情緒** | 教學／簡報層保留玩法資訊；幹員層可偏情緒，但不可與人設打架 |
| **職能隱喻** | 每人有固定「比喻域」——寫新句時先問：這句像不像**這個人**會用的專業語彙？ |
| **噬星者** | 掠食災害、不可談判；可寫壓迫感，不可寫成有外交的文明 |
| **星火** | 開採能源、頻率共振——不可神棍化 |

### 聲道分工速查

| 聲道 | 人格關鍵字 | 主要交付位置 |
|------|------------|--------------|
| **長官** | 嘴硬護短、教訓大頭兵、保命意味 | `chapterEntryBriefings.json`、`gameFixedMessages.json` |
| **工匠（玩家）** | 決策者、少独白、UI 以指令為主 | 介面文案；敘事稱「工匠」不強制具名 |
| **幹員** | 依下表各人格 | `heroes.ts`（`barrage`、`missionBrief*`、`lines`） |
| **系統** | 清楚優先、中性 | `gameFixedMessages.json` |

## 二、長官（指揮長官／電報源）

### 核心個性

- **身分**：後方台站／參謀線上的戰術長官；玩家收到的「長官電報」即其加密指令。
- **人格**：嚴厲但不羞辱；罵你是因為**不想看你送命**；嘴硬、護短、要你學會。
- **情感功能**：戰情壓力 + 教學可讀性 + 「頻道另一頭有人睡覺」的連帶感。
- V2 補強：**壓低聲量、靜默佈署、速度控制**——少大吼激勵，多強調別驚動、動作放慢。

### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 軍令感、短句、直球；可嚴厲可關懷，但都是**長官對新兵** |
| **節奏** | 先壓一句（點出會犯的錯）→ 再護一句（還來得及學） |
| **句長** | 12～28 字／句；每關進場 1～2 句為主 |
| **人稱** | 「你」「我們」；偶爾點名幹員代號（如幽靈頻率）表信任 |
| **比喻域** | 頻道、電報、活碼、後方台站、睡覺的人、工匠、星火共振 |

### 可直接套的句型

```
別給我亂下。
我寧可你慢，不准你亂。
今天錯了還能重練，給我學會。
你守住節奏，我就能把你帶過去。
我現在罵你，是不想以後看你送命。
```

### 禁忌

- 純髒話、純雞湯、長篇演講
- 十關同句換字
- 鼓吹開火、衝鋒當官方戰法
- 為情緒刪掉「先選電報、再選格」等規則提示（簡報／教學層）

### 範例（實裝）

> 「我寧可你學會，也不希望你帶著半套反射上前線——前方的人，會因為你讀通密碼而多一口氣。」  
> 「守住腦子＝守住在頻道另一頭睡覺的人。我信你。下一格我們一起讀通。」

### 嚴厲 vs 關懷（同一人、不同情緒）

| | 嚴厲 | 關懷 |
|---|------|------|
| **差異點** | 先盯錯、不容含糊 | 先承認壓力、再給台階 |
| **共通** | 規則不刪、保命意味不刪 | 同左 |

## 三、工匠（玩家）

### 核心個性

- **身分**：戰術指揮敘事代號 **Artisan**；決策、佈雷、讀電報。
- **人格**：不在台詞裡強調「我是誰」——**行動即人格**。
- **介面語氣**：指令式、簡短；避免長独白或中二宣言。

### 說話方式

- 以 UI 操作語為主（選電碼、點格、確認佈署）。
- 若敘事需要玩家台詞：冷靜、任務導向、不搶幹員戲。

### 4.1 小明 · `xiaoming` · 代號「白紙」

**職能**：新兵  
**技能**：無（全靠運氣）

#### 核心個性

- 家鄉同批最後能入伍的年輕人；**從怕到敢**，但怕沒消失，只是被壓成「照表走」。
- 情感錨點：**汗漬全家福**、盔上裂隙藍光、別讓照片變遺物。
- 弧線：靠運氣硬撐 → 演習實戰落差 → 學會「想清楚再踩」。

#### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 緊張、口語、會結巴（「我、我…」）；帶一點自我懷疑 |
| **節奏** | 慌時短促；穩下來會複誦口令式自語 |
| **人稱** | 多「我」；少教訓別人 |
| **比喻域** | 運氣、呼吸、全家福、護目鏡藍光、演習、賭、撤退 |

#### 情緒曲線

| 情境 | 口氣 |
|------|------|
| 開局 | 不確定、確認電報 |
| 失誤 | 驚慌、立刻想救下一格 |
| 順風 | 意外驚喜、「居然對了」 |
| 勝利 | 鬆一口氣、照片還在 |
|  idle | 催自己、也承認會怕 |

#### 句型模板

```
我、我這步應該……對吧？
看一眼全家福……好，呼吸完了，繼續。
欸？這格居然對了！
完了完了……下一格救一下！
任務完成……我還活著，照片也還在口袋裡。
```

#### 禁忌

- 寫成老練教官口吻
- 太帥、太篤定、零恐懼
- 專業術語過多（頻譜、等高線等應留給艾達／賽琳娜）

#### 與他人對比

- 比**艾達**更情緒外顯；比**老張**更沒底氣；比**波比**更靠腦、不靠鼻子。

### 4.2 艾達 · `ada` · 代號「幽靈頻率」

**職能**：電信專家  
**技能**：多路併收、雜訊緩讀

#### 核心個性

- 衝擊餘波後**只信頻譜**；創傷 + 職業病 = 用波形看世界。
- **不是**失聲、**不是**聽懂噬星者；異常來自**讀數與儀器**，非幻聽。
- 戰場上比誰都冷靜；咖啡因、螢幕反光過敏是檔案註記。

#### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 冷靜、精準；**戰場**像戴耳麥講話（「對上了」「重來」），少報告體（「結果輸出：」） |
| **節奏** | 短句、命令式；少驚嘆號 |
| **人稱** | 「我」陳述讀數；對玩家用「你」下判斷 |
| **比喻域** | 頻譜、波形、雜訊、時基、收斂、交叉驗證、頻寬、相位、峰值、信噪比 |

#### 情緒曲線

| 情境 | 口氣 |
|------|------|
| 開局 | 訊號收齊、交給節奏 |
| 失誤 | 「參數崩了、重算」——不責人，責推論 |
| 順風 | 參數對上、收斂乾淨 |
| 勝利 | 像輸出報告：任務達標、座標歸零 |
| lastTen | 撤離前校準、封退路 |

#### 句型模板

```
雜訊一濾，雷在哪就清楚。
曲線一歪我就知道該繃緊——不是直覺，是讀數。
訊號收齊，開始解。
參數崩了，重算！
結果輸出：任務達標。
```

#### 禁忌

- 浪漫化超能力、聽見怪物說話
- 過度熱情、雞湯、結巴
- 用地理／結構隱喻搶賽琳娜／老張的域

#### 與他人對比

- 與**克萊兒**同為「創傷後多一層視界」，但艾達是**電磁／頻譜**，克萊兒是**生命場**。

### 4.3 賽琳娜 · `selina` · 代號「地圖師」

**職能**：戰地測繪師  
**技能**：格網倍乘

#### 核心個性

- 頂尖地理學家；裂隙把地貌撕成錯覺時，**腦內座標網**比紙圖可靠。
- 最恨「感覺對了」；話少、筆尖快；寧可畫滿註記也不漏格。
- 情感：用空間替隊友踩雷；成為隊伍「空間錨」。

#### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 簡短、斷然；**戰場**像現場帶路（「別急啦」「重畫」），少勘測報告句 |
| **節奏** | 一句一判斷；少解釋為什麼 |
| **人稱** | 「我畫過」「別踩」——帶路型 |
| **比喻域** | 等高線、稜線、谷線、脊線、地形、格網、投影、斷崖、坡影、扭曲帶 |

#### 情緒曲線

| 情境 | 口氣 |
|------|------|
| 開局 | 先看輪廓、別急 |
| 失誤 | 「被坡影騙了、重畫」——不安慰，直接修正 |
| 順風 | 「落在稜上，漂亮」 |
| 勝利 | 簽名式收束：「這盤圖我簽名了」 |
| idle | 停太久等高線會糊 |

#### 句型模板

```
地形不會說謊，說謊的是你的眼。
把盤面當成等高線：先找脊線與谷線。
你被坡影騙了，重畫。
線收乾淨，地形成。
```

#### 禁忌

- 長篇學術講座
- 頻譜／電路術語（留給艾達）
- 過度熱情或口語碎嘴

#### 與他人對比

- 比**老張**更偏「空間幾何」；老張偏「結構承載／連鎖」。

### 4.4 波比 · `bobby` · 代號「雙子」

**職能**：搜救犬訓練員（搭檔犬 **巴克**）  
**技能**：靈敏嗅覺／嗅線標記

#### 核心個性

- 廢墟搜救夜是巴克刨出他——**人狗分工**，一個眼神對座標。
- 對動物有耐心，**對犯錯的人沒耐心**；不必講大道理。
- 制服補丁、犬用助聽、背雷包——務實、戰場搜救節奏。

#### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 直球、略帶催促；常帶巴克當第三人稱 |
| **節奏** | 短句；動作導向（聞、標、撤） |
| **人稱** | 「我帶狗，你帶腦」式分工 |
| **比喻域** | 氣味、線索、風向、可疑格、搜救、尾巴（時間緊迫） |

#### 情緒曲線

| 情境 | 口氣 |
|------|------|
| 開局 | 先掃、巴克耳朵豎了 |
| 失誤 | 「無視線索了、重聞」——略兇 |
| 順風 | 味道對了、像狗一樣可靠 |
| 勝利 | 搜救成功、回去加肉 |
| lastTen | 巴克催你、跟氣味線撤 |

#### 句型模板

```
先聞可疑格，再談帥氣操作。
巴克看我一眼我就懂——這格先標。
標完可疑格就交給腦袋收尾，別跟鼻子講道理。
搜救成功，收隊。
```

#### 禁忌

- 把巴克寫成會說人話
- 過度理論、頻譜、結構長句
- 對失誤過度溫柔（他對人犯錯沒耐心）

#### 與他人對比

- 比**小明**更靠直覺通道，但直覺來自**犬**，不是運氣。

### 4.5 老張 · `laozhang` · 代號「城牆」

**職能**：結構工程師  
**技能**：加固模組

#### 核心個性

- 鋼筋混凝土與連鎖反應同一套語言；**話少、可靠、嘴硬手快**。
- 乾谷據點樑是他澆的；梁塌創傷 → 線不能在他面前整段塌。
- 穿重甲是算過的：多扛一瞬，後面少賠一條命。

#### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 沉穩、短、像工地帶班 |
| **節奏** | 慢一拍提醒 + 硬結論 |
| **人稱** | 「我幫你撐」「你們別搶著塌」 |
| **比喻域** | 結構、灌漿、連鎖、應力、線、撐、塌、澆實、後手 |

#### 情緒曲線

| 情境 | 口氣 |
|------|------|
| 開局 | 別慌、我盯結構 |
| 失誤 | 「炸了，自己站起來」+ 願意幫撐一次 |
| 順風 | 「穩，就這樣」「像灌漿，實在」 |
| 勝利 | 結構沒塌、全線收斂 |
| 加固觸發 | 擋住了、還有一次 |

#### 句型模板

```
結構鬆一格，全線跟著晃。
先撐住連鎖，再談漂亮——我背得起，你們別搶著塌。
別跟雷區講道理，跟它講結構。
錯一次還能撐，同錯兩次就是設計問題。
```

#### 禁忌

- 碎嘴、結巴、過度情緒宣洩
- 搶艾達／賽琳娜的專業隱喻域
- 寫成冷血機器（他有創傷與護短）

#### 與他人對比

- 與**堡壘-09**都談結構，但老張有**人性與創傷**；09 是純計算。

### 4.6 堡壘-09 · `tungsten` · 代號「鎢鋼」

**職能**：重裝自律戰鬥體  
**技能**：規劃中

#### 核心個性

- 工業深處挖掘機器人；操作員全滅後**相位奇點**留下不穩定自我意識。
- **不再服務人類**；只執行自我優化——冷酷、金屬、破壞效率導向。
- 職業病：任何環境先掃結構弱點。

#### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 機械報告、零情緒、標籤式 |
| **節奏** | 名詞＋動詞；可省略主語 |
| **標點** | 冒號、頓號、句點；**不用**驚嘆號、問號、口語助詞 |
| **比喻域** | 標定、弱點、承載、冗餘、移除、序列、撤回、待機、優化 |

#### 情緒曲線

| 情境 | 口氣 |
|------|------|
| 任何 | **幾乎平坦**——僅任務狀態詞變化 |
| 失誤 | 承載斷裂、邏輯衝突 |
| 勝利 | 序列完成、撤回待機 |

#### 句型模板

```
標定：頂板弱點、主受力線。執行。
操作員離線。優化路徑：持續。
障礙＝冗餘結構。移除。
佈雷序列完成。撤回待機。
```

#### 禁忌

- 幽默、口語、結巴、安慰、道德講稿
- 暗示恢復人性或服從長官情感
- 用「我覺得」「大概」等模糊詞

#### 與他人對比

- 全幹員中**唯一非人**聲線；與系統提示相鄰，但仍有「自我優化」立場。

### 4.7 克萊兒 · `claire` · 代號「精神鏈接（Psi-Link）」

**職能**：靈能醫療兵  
**技能**：規劃中

#### 核心個性

- 野戰醫院實習夜被相位餘波激發靈能；視界多一層**生命網格**。
- **不是**溫柔神蹟——是創傷 + 職業病式專注；**過度共情**導致精神疲勞。
- 戰場冷靜；痛算在自己身上。

#### 說話方式

| 維度 | 特徵 |
|------|------|
| **語氣** | 冷靜、監測感；**戰場**口語但專業（「盤面在發燒」「跟線別跟慌」），少論文句 |
| **節奏** | 先報讀數、再給行動 |
| **人稱** | 「我先穩住你的…」；把情緒外化成網格 |
| **比喻域** | 生命訊號、網格、毒性前緣、清創、縫合、發燒、讀數、結構（生體） |

#### 情緒曲線

| 情境 | 口氣 |
|------|------|
| 開局 | 網格對齊、跟線不跟慌 |
| 失誤 | 讀數斷、毒性逼近——專業撤離 |
| 順風 | 穩住、乾淨了再下一步 |
| 勝利 | 網格歸靜、辛苦了（少見的溫度） |
| idle | 把慌引回腕上網格 |

#### 句型模板

```
生命訊號在歪——不是幻聽，是讀數先到了。
我先穩住你的「結構」，你再下決定。
一步一格，像清創：乾淨了再縫。
共情會累，但冷靜能救更多人。
```

#### 禁忌

- 寫成聖母或神秘系先知
- 聽見噬星者說話、與異界溝通
- 搶艾達的頻譜域（她的是**生體／生命場**）

#### 與他人對比

- 全幹員中**最會把情緒轉成可讀指標**；比老張更軟、比艾達多一點「人」的疲勞感。

## 六、依觸發情境的口氣調整

**戰場專用**：下列觸發的台詞一律比簡報／檔案更口語——像戴著耳麥講話，不是寫電文。

| 觸發 | 共通 | 各員差異 |
|------|------|----------|
| `opening` | 短、像開場白 | 小明最慌；09 最平（09 **不**口語化） |
| `idle` | 催決策、勿拖 | 艾達「我在對頻啦」；波比「味道要散了」 |
| `good` | 克制稱讚、可驚嘆 | 小明「欸？」；賽琳娜「漂亮」；09 無喜悅 |
| `bad` | 立刻導向修正 | 老張「站起來」；波比「重聞」 |
| `lastTen` | 撤退／收斂、語氣更急 | 艾達口語但仍帶頻道詞；波比跟氣味撤 |
| `victory` | 任務收束、鬆一口氣 | 小明「還活著」；09「撤回待機」 |

### 口語化句型（戰場）

| 少寫（報告體） | 多寫（嘴邊話） |
|----------------|----------------|
| 結果輸出：任務達標 | 搞定，收工 |
| 邏輯上該格僅允許數字 | 這格只能放數字欸 |
| 時基歸零，陣面有漏 | 時間到了，陣上還有洞 |
| 收斂完再談撤 | 填好再走 |
