# Emotion Route Balance

目標：把每關長官台詞做成「嚴厲 / 關懷」兩條情感路線，或統一成「長官教訓大頭兵」風格，提升代入感，同時不犧牲教學可讀性與節奏。

## 快速路由

- 第一關雙路線可直接套用稿：`ref/lv1-tone-pack.md`
- 是否加更多對話與加在哪：`ref/dialogue-expansion-checklist.md`
- 章內批量改寫時的口氣規則：`ref/sergeant-tone-rules.md`

## 最小交付（每次調整都要做到）

- 指定關卡（預設 Lv1）提供嚴厲版與關懷版各一組台詞。
- 明確標示更新位置（`chapterEntryBriefings.json` 或 `gameFixedMessages.json` 鍵位）。
- 每組台詞都保留核心規則提示：先選電報數字、再選座標、失誤可重練。
- 若是批量改一整章，先統一該章口氣，再讓每關各有一句自己的壓力點，不可十關都像同一句改字。
- 若使用者要「提高遊戲性」，至少再補 1 組情境反應句（勝利或失敗）。

## 操作準則（先 Lv1）

- Step 1：先讀目前 Lv1 原文，抽出不可丟失的規則訊息。
- Step 2：產出兩種語氣，差異放在情緒與關係，不改機制說明。
- Step 3：補一條動態回饋句（例：首次失誤時改成關懷提醒）。
- Step 4：回覆時附上可直接貼上的 JSON 片段。

## Gotchas（必保留）

- 只改情緒，不提規則，會讓新手看不懂要怎麼操作。
- 全是長句訓話，沒有短反饋句，節奏會拖慢。
- 嚴厲版與關懷版只有換形容詞，沒有關係差異，玩家感受不到分支。
- 同時改太多關，無法快速驗證文案是否提升留存。
- 把盤面上看得到的資訊（邊界大小、可部署格數、顏色提示）重複寫進簡報或指南，會讓玩家直接略過。
