# 可用音效來源與授權摘要

以下是常用且可用於遊戲製作的來源；實際使用前仍需逐包確認當下 license 條款。

## 1) Sonniss - GameAudioGDC Bundle

- 連結：<https://sonniss.com/gameaudiogdc/>
- 特點：每年大量高品質素材包，常見於專業遊戲/影視工作流。
- 授權摘要：一般可商用、可終身使用於專案；不可轉售素材本身；注意 AI/ML 訓練限制條款。

## 2) OpenGameArt

- 連結：<https://opengameart.org/>
- 特點：社群資源多，含 SFX/BGM/素材包，適合快速原型與低預算案。
- 授權摘要：不同資產授權不同（CC0、CC-BY、GPL 等），必須逐項記錄與對應署名需求。

## 3) Kenney (含 OGA 鏡像項目)

- 連結：<https://opengameart.org/content/kenney-assets>
- 特點：遊戲開發常用素材提供者，風格一致，適合 UI/休閒遊戲。
- 授權摘要：多數為寬鬆授權，但仍以官方頁面最新條款為準。

## 4) 音效設計文件參考（不是素材庫）

- Audio Design Doc 教學：
  - <https://zq-sfx.com/audiodesigndoc>
  - <https://www.asoundeffect.com/game-audio-design-document/>
  - <https://thiagoschiefer.com/home/documentation-and-organization-in-game-audio-with-templates>
- 用途：拿來定義你的專案音效藍圖、命名、任務拆分與驗收標準。

## 來源選用建議（短版）

- 要高品質大量素材：先看 Sonniss。
- 要快速湊齊原型：OpenGameArt/Kenney。
- 要長期維護：先做命名與盤點，再進素材，避免後續返工。
